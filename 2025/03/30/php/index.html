表达式

表达式是 PHP 中最基本的构建块，它是由值、变量、运算符和函数调用组合而成的一段代码，能计算出一个值。PHP 中的表达式可以是简单的字面量，也可以是复杂的嵌套结构。

1. 字面量表达式

字面量是直接表示值的表达式，常见的有：

  • 整数：如 10、-20 。
  • 浮点数：如 3.14、-0.5 。
  • 字符串：可以用单引号或双引号括起来，例如 'hello'、"world" 。
  • 布尔值：true 和 false 。
  • 空值：null 。

PHP弱比较

在 PHP 里，弱比较是一种在比较两个值时只关注值本身，而不严格比较数据类型的比较方式。这和强比较（全等比较）不同，强比较不仅要求值相等，还要求数据类型相同。下面将详细介绍 PHP 弱比较的相关内容。

弱比较运算符

PHP 中使用 == 进行弱比较，使用 === 进行强比较。以下是弱比较和强比较的基本示例：

不同数据类型的弱比较规则

1. 数值和字符串比较

当一个数值和一个字符串进行弱比较时，PHP 会尝试将字符串转换为数值，然后再进行比较。

在上述代码中，$str1 转换为数值是 10，$str2 从字符串开头开始转换为数值也是 10，所以它们和 $num 弱比较结果为 true；而 $str3 从字符串开头无法转换为有效的数值，转换结果为 0，所以和 $num 弱比较结果为 false。

2. 布尔值和其他类型比较

布尔值和其他类型比较时，true 会被当作 1，false 会被当作 0。

3. 数组比较

数组的弱比较会比较数组的键值对，只有当两个数组的键值对完全相同（不考虑顺序）时，才会认为相等。

4. 对象比较

对象的弱比较会比较对象的类和属性，如果两个对象是同一个类的实例，并且所有属性的值都相等，则认为它们相等。
